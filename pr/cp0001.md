# Nekoniumをもらってコミケで投げようキャンペーン

Androidモバイルウォレットをリリースしましたので、動作テストも兼ねて仮想通貨Nekonium(NUKO)の配布キャンペーンを実施します。
イベントなどでチップの代わりに利用していただければ幸いです。

# 参加方法

NUKOチップをあげる人ﾁｰﾑとNUKOチップをもらう人ﾁｰﾑのどちらかでご参加ください。

## NUKOをあげる人ﾁｰﾑ
一般参加者の方を想定しています。


1. Nekonium mobile wallet(モバイルウォレット)をインストールしてください。
2. Twitterでハッシュタグ<b>#nekonium</b>をつけて、QRコード画面のスクリーンショットを送信してください。
3. 中の人から111NUKOもらえます。
4. モバイルウォレットで受信を確認したら、下記にあるテスト用QRコードをカメラでスキャンして、1NUKOをテストアドレスに送信する練習をしてください。
5. イベント会場でNUKOのQRコードを見つけてウォレットからチップを送信してください。

- 会場で消費できなかったNUKOは差し上げます。

## NUKOをもらう人ﾁｰﾑ
コミケに出展される方、コスプレイヤーの方を想定しています。

1. QRコードジェネレータ、もしくは外部ツールでNekoniumアドレスのQRコードを作成してください。
2. Twitterにハッシュタグ<b>#nekonium</b>をつけて、QRコードの画像、もしくはQRコードが確認できる写真投稿してください。
3. 中の人から練習用の111NUKOが送られてきます。
4. モバイルウォレットを使う場合は、テスト用QRコードをカメラでスキャンして、1NUKOをテストアドレスに送信する練習をしてください。
4. 作ったQRコードを印刷して、イベント会場で提示、もしくは掲示してください。

- 使用するNekoniumアドレスは、MyNukoWallet,Mistウォレト,モバイルウォレットのどれでも構いません。
- 取得したチップ、及び配布したNUKOはすべて差し上げます。

# 物販決済にNekoniumを使う予定の方へ

+ 現在のねこにうむパブリックノードサーバーはぴよぴよです。サーバーダウンが発生する可能性があります。
+ その場での決済に失敗することが考えられます。必ず他の決済手段を併用してください。
+ Nekonium ﾃﾞﾌﾞチームのNekoniumの価値に対する認識は無価値です。対価の価値はご自身で判断してください。

# イベントにいかないけどキャンペーンに参加したい方へ

NUKOをあげるひとはどなたでも参加できます。
NUKOをもらうひととして参加する場合は、QRコードをご自身に著作権のある創作物の写真に張り付けて投稿してください。
創作物には、絵画、電子工作、謎の自作物体などを含みます。ブログについては投資を煽る内容のものは除外します。
